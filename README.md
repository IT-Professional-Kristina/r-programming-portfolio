# R Programming Portfolio

A progressive, project-based R programming curriculum built from zero 
exposure, themed around cosmology, space science, and the works of 
Stephen Hawking.

## Purpose

This repository documents my R programming journey as part of my broader 
healthcare IT and clinical informatics portfolio. Each script builds on 
the last, introducing new concepts through real astronomical data and 
cosmic constants.

## Scripts

### cosmic_constant.R
- Introduction to R as a calculator
- Variable assignment using the `<-` operator
- Scientific notation in R
- Calculations using named cosmic constants including speed of light,
  light year distance, age of the universe, and Planck's constant
- First debugging experience: silent wrong value from missing `e` 
  in scientific notation

### star_profile.R
- R data types: numeric, character, logical, integer
- Using `class()` to inspect variable types
- Type conversion with `as.numeric()`
- Real-world application: how type mismatches appear in 
  healthcare data imports

### star_catalog.R
- Vector creation with `c()`
- Statistical functions: `mean()`, `max()`, `min()`, `length()`
- Vector indexing with `[]`
- Vector math operating across entire datasets simultaneously
- Logical comparisons across multiple observations

## Skills Demonstrated
- R syntax and data types
- Variable assignment and naming conventions
- Scientific notation handling
- Debugging and error interpretation
- Vector operations and analysis
- Code commenting best practices

## Learning Context
Scripts written in RStudio using R 4.6.0 as part of a structured,
module-based curriculum progressing toward healthcare data analysis,
clinical informatics, and genomic data workflows in R.
